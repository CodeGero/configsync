# configsync

> Sync configuration across environments safely.

[![PyPI](https://img.shields.io/pypi/v/kryptorious-configsync)](https://pypi.org/project/kryptorious-configsync/) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Install

```bash
npm i -g kryptorious-configsync
```

## Quickstart

```bash
npm i -g kryptorious-configsync
configsync diff staging prod
```

## Commands

| Command | Description |
|---------|-------------|
| `configsync push` | Push local config to a target environment. |
| `configsync diff env1 env2` | Show config differences between environments. |
